# 🛒 Amazon Clone (HTML & CSS)

## 📌 Description
This project is a static frontend clone of the Amazon homepage built using HTML and CSS.  
It focuses on replicating a real-world e-commerce interface to understand layout design, styling, and structured UI development.

---

## 🚀 Features
- Amazon-inspired homepage layout
- Navigation bar with logo, search bar, and menu options
- Product sections and promotional banners
- Structured footer similar to Amazon
- Clean and well-organized UI design

---

## 🛠️ Tech Stack
- HTML
- CSS (Flexbox, Grid)

---

## 📚 What I Learned
- Structuring and designing large-scale web pages
- Applying Flexbox and Grid for efficient layout management
- Improving UI/UX design understanding
- Replicating real-world website interfaces
